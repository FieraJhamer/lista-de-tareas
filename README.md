# Lista de Tareas

Aplicación web simple de lista de tareas (To-Do List) desarrollada con **HTML, CSS y JavaScript puro**.

> [!NOTE]  
> No se requiere ninguna dependencia, sólamente descargar el archivo .html y ejecutarlo

## Características

-  Agregar y eliminar tareas
-  Marcar tareas como completadas
-  Reordenar tareas con drag & drop
-  Persistencia en `localStorage`
-  Interfaz mínima
